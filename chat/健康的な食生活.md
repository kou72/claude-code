# 体制ビジネス：構造全体図

![体制ビジネス：構造全体図](charts/taisei_structure.png)

## 色の凡例

### 赤・実線：お金の流れ

| # | 経路 | 内容 |
| --- | --- | --- |
| ① | 案件企業 → リーダーの自社 | 体制単価（月次一括） |
| ② | リーダーの自社 → リーダー | リーダー本人の稼働単価 |
| ③ | リーダーの自社 → BP×N | BP単価（マージンを抜いた後） |

### 破線：情報の流れ（色別）

| 色 | 経路 | 内容 |
| --- | --- | --- |
| 青 | 営業担当 ↔ 案件企業 → 自社 | 案件情報の取得・持ち込み |
| 紫 | 案件企業 ↔ リーダー | 要件・評価のやりとり |
| 緑 | リーダー → 自社 | 進捗・請求根拠の報告 |
| オレンジ | リーダー ↔ BP | 指示・実績のやりとり |

## 構造から見える問い

1. **営業担当は誰か**：自社内の営業か、外部エージェントかで情報の入口が変わる
2. **リーダーと自社の関係**：社員か個人事業主かで契約構造が変わる
3. **情報の断絶点**：リーダー→自社への「進捗・請求根拠」が今どうやって連携されているか（口頭？メール？Excel？）
