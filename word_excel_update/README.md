# Word × Excel 自動更新ツール

Excelに変更内容を入力して「Word を更新」ボタンを押すだけで、
Wordの指定箇所が**赤字**で自動更新される仕組みです。

## ファイル構成

```text
word_excel_update/
  setup.ps1              # Excelファイル（.xlsm）を自動生成するセットアップスクリプト
  WordUpdater.bas        # Excelに組み込むVBAマクロ本体
  WordBookmarkHelper.bas # Wordに組み込むブックマーク管理ヘルパー
  README.md              # このファイル
  Word更新ツール.xlsm    # ← setup.ps1 実行後に生成される
```

---

## セットアップ手順

### Step 1: Excelファイルを生成する

PowerShellを開き、このフォルダで以下を実行します。

```powershell
powershell -ExecutionPolicy Bypass -File setup.ps1
```

`Word更新ツール.xlsm` が生成されます。

> マクロのインポートに失敗した場合は、後述の「手動インポート手順」を参照してください。

### Step 2: WordにブックマークHelperを組み込む

1. 対象のWordドキュメントを開く
2. `Alt + F11` でVBAエディタを起動
3. メニュー「ファイル」→「ファイルのインポート」
4. `WordBookmarkHelper.bas` を選択してインポート
5. VBAエディタを閉じる（`Alt + F4`）

### Step 3: Wordドキュメントにブックマークを設定する

更新したい箇所にブックマークを設定します。

1. Wordで更新したいテキストを**ドラッグして選択**
2. `Alt + F8` → マクロ一覧から `WordBookmarkHelper.AddBookmark` を選択 → 実行
3. ブックマーク名を入力（例: `bm_company_name`）

#### ブックマーク名の規則

```text
✅ 半角英字・数字・アンダースコアのみ
✅ 先頭は英字推奨

例:
  bm_company_name   ← 会社名
  bm_contract_date  ← 契約日
  bm_amount         ← 金額
  bm_tantou_name    ← 担当者名
```

> **既存のブックマーク一覧の確認:** `WordBookmarkHelper.ListAllBookmarks` マクロを実行

## 日常的な使い方

### 1. Excelに変更内容を入力する

`Word更新ツール.xlsm` を開き、以下のように入力します。

| 列 | 内容 | 例 |
| -- | ---- | -- |
| A列 | ブックマーク名（Wordと一致させる） | `bm_company_name` |
| B列 | 説明（任意・自分用メモ） | `会社名` |
| C列 | **変更後のテキスト** | `株式会社新しい商事` |
| D列 | 状態（自動入力） | 更新後に「済」と表示 |

**B1セル**にWordファイルの絶対パスを入力します。

```text
例: C:\Users\username\Documents\2026\納品物_顧客名.docx
```

### 2. 「▶ Word を更新」ボタンをクリックする

- Wordが自動で開き（または前面に出て）、ブックマーク箇所が**赤字**に更新されます
- 更新された行のD列が **「済」（緑）** になります
- Wordファイルは自動保存されます

### 3. 納品前に赤字を黒字に戻す

最終納品前に赤字を黒字に戻す場合は、Wordで以下のマクロを実行します。

`Alt + F8` → `WordBookmarkHelper.ResetAllRedToBlack` → 実行

---

## ボタン一覧（Excel）

| ボタン | 動作 |
| ------ | ---- |
| ▶ Word を更新 | C列のテキストでWordを更新（D列が「済」の行はスキップ） |
| ↺ 状態リセット | D列の「済」を全てクリア（再度全行を更新対象にする） |
| ⬇ ブックマーク一覧取得 | WordのブックマークをExcelに一覧として取り込む |

## よくある質問とトラブルシューティング

### ブックマークが見つかりません と表示される

- Excel A列のブックマーク名とWord側のブックマーク名が**完全一致**しているか確認
- Wordで `WordBookmarkHelper.ListAllBookmarks` を実行してブックマーク名を確認
- スペルミスや全角/半角の違いがないか確認

### マクロが実行できない（セキュリティエラー）

Excelのセキュリティ設定でマクロが無効になっています。

1. `ファイル` → `オプション` → `セキュリティセンター`
2. `セキュリティセンターの設定` → `マクロの設定`
3. 「警告を表示してすべてのマクロを無効にする」を選択し、ファイルを開いた際に「コンテンツを有効にする」をクリック

### Wordが開かない / 「ファイルが見つかりません」エラー

- B1セルのパスに**バックスラッシュ**（`\`）が使われているか確認
- ファイルが移動・削除されていないか確認
- パスをエクスプローラーのアドレスバーからコピーして貼り付けると確実

### setup.ps1 実行時に「VBProject へのアクセスが拒否されました」エラー

ExcelのVBAプロジェクトアクセス設定が必要です。

1. Excelを起動 → `ファイル` → `オプション` → `セキュリティセンター`
2. `セキュリティセンターの設定` → `マクロの設定`
3. 「VBA プロジェクト オブジェクト モデルへのアクセスを信頼する」にチェック

## 手動インポート手順（setup.ps1 が使えない場合）

1. Excel を起動し、新しいブックを作成
2. シート名を「変更箇所」に変更
3. `Alt + F11` でVBAエディタを起動
4. メニュー「ファイル」→「ファイルのインポート」
5. `WordUpdater.bas` を選択してインポート
6. ファイルを `.xlsm`（マクロ有効ブック）として保存
7. シートにボタン（図形）を追加し、マクロ `WordUpdater.UpdateWordDocument` を割り当て

## 仕組みの概要

```text
[Excel: Word更新ツール.xlsm]          [Word: 納品物.docx]
┌─────────────────────┐               ┌────────────────────────┐
│ B1: Wordファイルパス │               │                        │
│                     │               │ ブックマーク箇所:      │
│ A列: ブックマーク名 │  VBAマクロで  │  {bm_company_name}     │
│ C列: 変更後テキスト │ ─────────────►│  {bm_date}             │
│                     │  自動更新     │  {bm_amount}           │
│ [▶ Word を更新]     │               │                        │
└─────────────────────┘               │ → 変更箇所が赤字になる │
                                      └────────────────────────┘
```
